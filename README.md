Uplink for PSP
===========

![Screenshot](screenshot.jpg)

Small homebrew game written in Lua for PSP 


##What's this ?
An unfinished clone of *Uplink Hacker Elite* game, written in Lua for PSP.

##Notes
Compatible with PSP Fat and Slims running GENB, M33 custom firmwares

##Installation Path
ms0:/PSP/GAME/
